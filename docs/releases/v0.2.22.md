# VAOL v0.2.22 Release Notes

Release date: 2026-02-23

## Highlights

- Added first-class gRPC ledger server support in `pkg/grpc` with parity coverage for core append, query, proof, checkpoint, verify, and export operations.
- Added Python SDK instrumentation wrappers for Anthropic and LiteLLM.
- Enabled dual-listener server startup (`--addr` for REST and `--grpc-addr` for gRPC) in `vaol-server`.
- Hardened startup and revocation paths with expanded tests for sparse sequence rebuild and checkpoint validation.
